Working with git

Befehl Beschreibung
atom . öffnet Atom im aktuellen Verzeichnis
git init Git Repo initialisiert
git status Zeigt den Status des git-repo (vor allem Dateien, die nicht im Git Repo sind und alle Dateien, die nicht "eingecheckt" sind)
git add . Fügt alle Dateien aus dem aktuellen Verzeichnis dem Git Repo zu (Punkt durch Dateinamen ersetzen für einelne Datei)
git commit -m "Message" "einchecken" mit Kommentar ("Message")
